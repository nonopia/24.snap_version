Basic pip install
  - (linux) apt install python3-pip / (windows) python.exe -m pip install --upgrade pip
  - 터미널 열때 기본적으로 수행하기 위해
    * Ctrl + Shift + P(or F1 key) → Python: Select Interpreter 선택
    * 가상 환경의 Python 경로 선택(예: _my_venv\Scripts\python.exe)

Windwows Virtual Environment 설치
  - * cmd 창에서 수행
  - py -m pip install --upgrade pip
  - python -m venv _my_venv
  - .\_my_venv\Scripts\activate

Linux Virtual Environment 설치
  - sudo apt install python3-venv
  - python3 -m venv _my_venv
  - source _my_venv/bin/activate

Environment Copy
  - pip list
  - pip freeze > req.txt
  - deactivate
  - python -m venv _my_venv
  - (linux) source _my_venv/bin/activate / (windows) .\_my_venv\Scripts\activate
  - pip install -r req.txt
  - pip list
  - (linux) rm req.txt / (windows) del req.txt
  - (linux) rm -r _old_venv / (windows) rmdir _old_venv 
'''
